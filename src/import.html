📥 Import Planning → Supabase

Migration d'un fichier planning.json local vers la base Supabase.

1Connexion

Connectez-vous avec un compte editor ou admin.

2Choisir le fichier planning.json

3Importer dans Supabase

Comportement :
  • Personnes existantes (même prénom + nom) → ignorées (pas de doublon).
  • Nouvelles personnes → insérées avec un UUID.
  • Étiquettes / ponts / réunions → insérés en remappant les références aux personnes.
